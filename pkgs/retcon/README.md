# `retcon`

## Description

`retcon` is a utility to search and bulk delete commands from your bash history using fzf. History is sorted by most recent first by default.

### Options

| Option            | Description                                                  |
| ----------------- | ------------------------------------------------------------ |
| `-h`, `--help`    | Show this help message and exit.                             |
| `-r`, `--reverse` | Display the history entries in reverse (least recent first). |
| `-v`, `--version` | Show the version of `retcon` and exit.                       |
