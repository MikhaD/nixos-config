# `retcon`

## Description

`retcon` is a utility to search and bulk delete commands from your bash history using fzf.

### Options

| Option            | Description                                                 |
| ----------------- | ----------------------------------------------------------- |
| `-h`, `--help`    | Show this help message and exit.                            |
| `-r`, `--reverse` | Display the history entries in reverse (most recent first). |
| `-v`, `--version` | Show the version of e and exit.                             |
